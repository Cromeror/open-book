## Descripcion

Crear la estructura de archivos para el modulo de Reportes y Dashboard con validacion de permisos configurada. Este scaffold sera el punto de partida para implementar OB-008 (Dashboard) y OB-009 (Reportes).

## Estructura de Archivos

```
apps/web/src/app/(dashboard)/reports/
├── page.tsx                    # Centro de reportes / Dashboard
├── contributions/
│   └── page.tsx                # Reporte de aportes por periodo
├── commitments/
│   └── page.tsx                # Reporte de compromisos vs cumplimiento
├── progress/
│   └── page.tsx                # Reporte de avance de objetivos
├── export/
│   └── page.tsx                # Exportar datos (PDF, Excel)
└── _components/
    └── .gitkeep
```

## Permisos por Pagina

| Pagina | Guard | Permiso |
|--------|-------|---------|
| /reports | requireModule | reports |
| /reports/contributions | requirePermission | reports:read |
| /reports/commitments | requirePermission | reports:read |
| /reports/progress | requirePermission | reports:read |
| /reports/export | requirePermission | reports:export |

## Criterios de Aceptacion

La logica de negocio se implementara en:

Formatos de exportacion:

- Excel/CSV para analisis

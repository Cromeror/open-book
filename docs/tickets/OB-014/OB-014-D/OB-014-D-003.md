## Descripcion

Crear la estructura de archivos para el modulo de Compromisos con validacion de permisos configurada. Este scaffold sera el punto de partida para implementar OB-006.

**NOTA**: Compromiso != Aporte. Un compromiso es una intencion de pago, no dinero real.

## Estructura de Archivos

```
apps/web/src/app/(dashboard)/compromisos/
├── page.tsx                    # Lista de compromisos
├── nuevo/
│   └── page.tsx                # Crear compromiso
├── [id]/
│   ├── page.tsx                # Detalle del compromiso
│   └── aportes/
│       └── page.tsx            # Ver aportes que cumplen este compromiso
└── _components/
    └── .gitkeep
```

## Permisos por Pagina

| Pagina | Guard | Permiso |
|--------|-------|---------|
| /compromisos | requireModule | compromisos |
| /compromisos/nuevo | requirePermission | compromisos:create |
| /compromisos/[id] | requirePermission | compromisos:read |
| /compromisos/[id]/aportes | requirePermission | compromisos:read |

## Criterios de Aceptacion

## Descripcion

Crear la estructura de archivos para el modulo de Actividades de Recaudo con validacion de permisos configurada. Este scaffold sera el punto de partida para implementar OB-005.

## Estructura de Archivos

```
apps/web/src/app/(dashboard)/actividades/
├── page.tsx                    # Lista de actividades
├── nueva/
│   └── page.tsx                # Crear actividad
├── [id]/
│   ├── page.tsx                # Detalle de actividad
│   ├── editar/
│   │   └── page.tsx            # Editar actividad
│   └── metas/
│       └── page.tsx            # Configurar metas
└── _components/
    └── .gitkeep
```

## Permisos por Pagina

| Pagina | Guard | Permiso |
|--------|-------|---------|
| /actividades | requireModule | actividades |
| /actividades/nueva | requirePermission | actividades:create |
| /actividades/[id] | requirePermission | actividades:read |
| /actividades/[id]/editar | requirePermission | actividades:update |
| /actividades/[id]/metas | requirePermission | actividades:manage |

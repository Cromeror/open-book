## Descripcion

Implementar navegacion responsiva con menu hamburguesa y drawer lateral para dispositivos moviles.

## Contexto Tecnico

### Archivos creados/modificados
- `apps/web/src/components/layout/Sidebar.tsx` - Sidebar responsivo con overlay
- `apps/web/src/components/layout/Header.tsx` - Boton hamburguesa
- `apps/web/src/components/layout/DashboardShell.tsx` - Estado de sidebar

### Enfoque tecnico
- Sidebar fijo en desktop (lg:translate-x-0)
- Drawer con overlay en mobile (transform transition)
- Boton hamburguesa en header para toggle
- Click en overlay cierra el sidebar

## Descripcion

Guiar al usuario para crear PQR de cambio de visibilidad con formulario simplificado.

## Criterios de Aceptacion

- [ ] Boton "Cambiar visibilidad" en perfil
- [ ] Redirige a formulario PQR pre-llenado
- [ ] Muestra visibilidad actual
- [ ] Explica que el cambio no es inmediato

## Notas de Implementacion

En perfil de usuario:
```
Visibilidad de Estados de Cuenta: PUBLICO

[Solicitar cambio a PRIVADO]

Nota: El cambio de visibilidad requiere una solicitud formal
que sera procesada por la administracion. El proceso puede
tardar hasta 15 dias habiles.
```

Al hacer clic, pre-llenar PQR:
```json
{
  "tipo": "PETICION",
  "categoria": "VISIBILIDAD",
  "asunto": "Solicitud de cambio de visibilidad a PRIVADO",
  "descripcion": "Solicito cambiar la visibilidad de mis estados de cuenta de PUBLICO a PRIVADO.",
  "metadatos": {
    "visibilidadActual": "PUBLICO",
    "visibilidadSolicitada": "PRIVADO"
  }
}
```

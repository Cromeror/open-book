## Descripcion

Incluir en el flujo de registro opciones claras para que el usuario elija la visibilidad de sus estados de cuenta, con explicacion de las implicaciones.

## Criterios de Aceptacion

- [ ] Checkbox/radio para elegir visibilidad
- [ ] Texto explicativo claro
- [ ] Implicaciones de cada opcion visibles
- [ ] Valor por defecto: PRIVADO (opt-in para publico)

## Notas de Implementacion

Texto sugerido:
```
Visibilidad de Estados de Cuenta

[ ] Deseo que mis estados de cuenta sean PUBLICOS

Si elige PUBLICO:
- Otros residentes que tambien eligieron PUBLICO podran ver sus estados de cuenta
- Usted podra ver los estados de cuenta de otros residentes PUBLICOS
- Este cambio no es inmediato y requiere solicitud formal para revertirlo

Si elige PRIVADO (predeterminado):
- Solo usted y la administracion podran ver sus estados de cuenta
- No podra ver los estados de cuenta de otros residentes

Esta decision cumple con la Ley 1581 de 2012 (Habeas Data).
```

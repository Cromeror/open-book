## Descripcion

Crear endpoint de detalle de actividad que muestre toda la informacion relevante para que un residente pueda participar, incluyendo instrucciones especificas segun el tipo.

## Contexto Tecnico

### Archivos afectados
- `src/api/routes/residentes.routes.ts`
- `src/api/controllers/residentes.controller.ts`
- `src/services/actividad.service.ts`

### Enfoque tecnico
Detalle enriquecido con instrucciones dinamicas segun el tipo de actividad.

## Criterios de Aceptacion

- [ ] GET /api/mis-actividades/:id implementado
- [ ] Verifica que la actividad pertenece a la copropiedad
- [ ] Incluye instrucciones segun tipo de actividad
- [ ] Muestra compromisos propios del residente (si existen)
- [ ] Muestra aportes propios del residente (si existen)

## Notas de Implementacion

Endpoint:
```
GET /api/mis-actividades/:id
```

Response:
```json
{
  "id": "uuid",
  "nombre": "Rifa electrodomesticos",
  "tipo": "RIFA",
  "descripcion": "Rifa con premios donados por comercios locales",
  "configuracion": {
    "precioBoleta": 10000,
    "premios": ["TV 50 pulgadas", "Nevera", "Lavadora"],
    "fechaSorteo": "2025-04-20"
  },
  "instrucciones": "Cada boleta tiene un valor de $10,000. Puedes comprar las boletas que desees. El sorteo se realizara el 20 de abril de 2025.",
  "objetivo": {
    "id": "uuid",
    "nombre": "Reparacion ascensores"
  },
  "miParticipacion": {
    "compromiso": {
      "id": "uuid",
      "monto": "50000.00",
      "estado": "PENDIENTE"
    },
    "aportes": [
      {
        "id": "uuid",
        "monto": "20000.00",
        "fecha": "2025-03-01"
      }
    ],
    "totalAportado": "20000.00",
    "pendiente": "30000.00"
  }
}
```

Instrucciones dinamicas por tipo:
- RIFA: precio boleta, fecha sorteo, premios
- DONACION: monto sugerido, mensaje
- CUOTA: monto y periodicidad
- EVENTO: fecha, lugar, precio entrada

## Descripcion

Crear endpoint de avance optimizado para residentes, mostrando progreso agregado sin revelar informacion de otros apartamentos.

## Criterios de Aceptacion

- [ ] GET /api/objectives-progress implementado
- [ ] Muestra objetivos de la copropiedad del residente
- [ ] Incluye avance agregado (porcentaje, faltante)
- [ ] NO incluye detalles de aportes individuales

## Notas de Implementacion

```
GET /api/objectives-progress
Authorization: Bearer <token-resident>

Response:
{
  "objectives": [
    {
      "id": "uuid",
      "name": "Reparacion ascensores",
      "description": "...",
      "progress": {
        "percentage": 65.0,
        "message": "Se ha recaudado el 65% del objetivo"
      },
      "myContribution": {
        "total": "150000.00",
        "percentageOfObjective": 1.0
      }
    }
  ]
}
```

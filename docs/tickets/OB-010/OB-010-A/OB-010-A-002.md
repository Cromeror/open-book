## Descripcion

Implementar endpoint para que residentes puedan crear nuevas solicitudes PQR.

## Criterios de Aceptacion

- [ ] POST /api/pqr implementado
- [ ] Solo residentes autenticados pueden crear
- [ ] Genera radicado automatico
- [ ] Asigna fecha limite segun tipo
- [ ] Notifica a administradores (via OB-013)

## Notas de Implementacion

Tiempos de respuesta:
- PETICION: 15 dias habiles
- QUEJA: 15 dias habiles
- RECLAMO: 15 dias habiles

Request:
```json
{
  "tipo": "PETICION",
  "asunto": "Cambio de visibilidad de estados de cuenta",
  "descripcion": "Solicito cambiar mi visibilidad de publico a privado...",
  "categoria": "VISIBILIDAD"
}
```

Response:
```json
{
  "id": "uuid",
  "radicado": "PQR-2025-00042",
  "tipo": "PETICION",
  "estado": "RADICADA",
  "fechaLimite": "2025-02-05",
  "mensaje": "Su solicitud ha sido radicada exitosamente"
}
```

## Descripcion

Implementar endpoint para que administradores respondan y cierren solicitudes PQR.

## Criterios de Aceptacion

- [ ] PATCH /api/pqr/:id/responder implementado
- [ ] Requiere respuesta escrita
- [ ] Cambia estado a RESPONDIDA
- [ ] Registra quien responde y cuando
- [ ] Notifica al solicitante

## Notas de Implementacion

Request:
```json
{
  "respuesta": "Su solicitud ha sido procesada. El cambio de visibilidad...",
  "cerrar": true  // Opcional, cierra directamente
}
```

Response:
```json
{
  "id": "uuid",
  "estado": "RESPONDIDA",
  "respuesta": "...",
  "respondidoPor": "Admin Lopez",
  "fechaRespuesta": "2025-01-20T10:00:00Z"
}
```

## Descripcion

Crear flujo especifico para solicitudes de cambio de visibilidad de estados de cuenta.

## Criterios de Aceptacion

- [ ] Categoria VISIBILIDAD reconocida
- [ ] Campos adicionales: visibilidadActual, visibilidadSolicitada
- [ ] Validacion de cambio valido (publico<->privado)

## Notas de Implementacion

Request especifico:
```json
{
  "tipo": "PETICION",
  "categoria": "VISIBILIDAD",
  "asunto": "Solicitud de cambio de visibilidad",
  "descripcion": "Solicito cambiar mi visibilidad de PUBLICO a PRIVADO porque...",
  "metadatos": {
    "visibilidadActual": "PUBLICO",
    "visibilidadSolicitada": "PRIVADO"
  }
}
```

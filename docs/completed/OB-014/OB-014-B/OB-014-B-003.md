## Descripcion

Crear la estructura de carpetas y paginas placeholder para todas las rutas del dashboard, organizadas por modulos.

### Enfoque tecnico
- Uso de Next.js App Router con Route Group `(dashboard)`
- Cada pagina verifica permisos de modulo usando `requireModule()`
- Pagina admin usa `requireSuperAdmin()`
- Redireccion a dashboard si no tiene permisos
